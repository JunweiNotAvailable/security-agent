# AI 安全警報分類系統 - 即時展示

這是一個即時展示頁面，展示 AI 驅動的安全警報分類系統與傳統人工分析的比較。

## 功能特色

- **即時 API 整合**：直接呼叫實際的分類 API
- **並排比較**：視覺化呈現人工 vs AI 的處理流程
- **模式偵測**：展示系統如何識別重複攻擊模式
- **繁體中文介面**：完整的中文使用者介面

## 使用方式

1. 啟動 FastAPI 伺服器：
   ```bash
   python run.py
   ```

2. 在瀏覽器中開啟：
   ```
   http://localhost:8000/demo/index.html
   ```

3. 選擇不同的警報情境並點擊「執行比較」按鈕

## 技術架構

- **前端**：React (透過 CDN)
- **後端**：FastAPI + OpenAI API
- **資料庫**：SQLite
- **AI 模型**：GPT-4o-mini

## 展示情境

1. **SSH 暴力破解攻擊** - 展示基本的威脅分類
2. **內部威脅 / 資料外洩** - 展示複雜的情境分析
3. **重複攻擊** - 展示模式偵測和自動升級功能
